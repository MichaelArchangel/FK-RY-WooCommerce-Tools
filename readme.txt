== Description ==

Let your WooCommerce store support ECPay / NewebPay / SmilePay.

Let your WooCommerce store more with Taiwan's habits.

== Installation ==

= Minimum Requirements =

* PHP 7.4.0+
* WordPress 5.6+
* WooCommerce 5.0+
